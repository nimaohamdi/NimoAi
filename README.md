## ✨ Features

- Clean, modern dark UI with excellent RTL support
- Real-time streaming responses
- Local chat history (saved in browser)
- Custom API settings (OpenAI, Groq, Local LLMs, etc.)
- Progressive Web App (PWA) — installable
- Fully responsive (mobile + desktop)
- No backend required

## 🚀 Quick Start

1. Go to the [Live Demo](https://nimaohamdi.github.io/NimoAi)
2. Click the settings icon (⚙️) and enter your API key
3. Start chatting with AI

## Supported Providers

Any **OpenAI-compatible** API:
- OpenAI
- Groq
- Together.ai
- Local servers (Ollama, LM Studio, etc.)

## Tech Stack

- HTML5 + CSS3 + Vanilla JavaScript
- Server-Sent Events (Streaming)
- LocalStorage
- PWA Support
